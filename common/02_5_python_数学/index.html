    数学函数

    数学相关函数

    数学相关函数大部分都在math模块当中，所以使用时要先导入math模块

        import math

    ceil() 对一个数值进行向上取整操作

        格式： math.ceil(数值)
    返回值：整型

    floor() 对一个数值进行向下取整操作

        格式: math.floor(数值)
    返回值:整型

    pow() 计算一个数值的幂运算。

        格式：math.pow(底数,指数)
    返回值：浮点型

    sqrt() 开平方运算

        格式：math.sqrt(数值)
    返回值：数值

    fabs() 绝对值运算

        格式： math.fabs(数值)
    返回值：一定是非负浮点数

    注意：在操作时如果操作的数值本身是整数，返回的数值依旧是绝对值计算之后的浮点数

    modf() 拆解数值为整数和小数的元组

        格式：math.modf(数值)
    返回值：（小数部分,整数部分）

    数学常量：

    pi  派

e  自然对数

    随机数模块 random

        import random

    random() 获取0-1之间的随机小数

        格式：random.random()
    返回值：随机0-1之间的小数

    choice() 随机返回序列中的某个值

        格式：random.choice(序列)
    返回值：序列中的某个值

    shuffle() 随机打乱列表

        格式：random.shuffle(列表)
    返回值：打乱顺序之后的列表

    uniform（） 返回指定范围内的随机数

        格式：random.uniform(开始值，结束值)
    返回值:随机返回开始和结束之间的小数。

    randrange() 随机返回指定范围内的指定间隔的数值

        格式：random.randrange([开始值,]结束值[,间隔值])
    返回值： 随机返回指定范围内的指定间隔的数值

    注意:默认开始值为0，默认间隔值为1

    数学相关内建函数

    round() 对一个数值进行四舍五入操作

        格式：round(数值)
    返回值：整型

    max() 获取一个数值序列中的最大值

        格式:max(数值相关的序列)
    返回值：序列中的最大值

    min() 获取一个数值序列中的最小值

        格式:min(数值相关的序列)
    返回值：序列中的最小值

    abs() 计算一个数值的绝对值

        格式:abs(数值)
    返回值： 数值的非负数数表现
